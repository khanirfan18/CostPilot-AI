# CostPilot AI

CostPilot AI is an intelligent financial agreement analyzer designed to uncover hidden fees, true costs, and effective APRs from complex loan documents and credit agreements. Built for transparency, it empowers users to truly understand what they are signing up for.

## Key Features
*   **AI-Powered Analysis:** Instantly parse unstructured text to extract loan amount, interest rates, and hidden fees.
*   **Multi-Model Support:** Seamlessly integrates with OpenRouter, Google Gemini, and OpenAI with a robust fallback mechanism.
*   **True Cost Calculation:** Calculates the actual effective APR and total payable amount, ignoring misleading marketed rates.
*   **Financial Trap Detection:** Highlights predatory clauses and hidden penalties before you sign.

## How it Works
1. Paste any financial agreement or loan document into the analyzer.
2. CostPilot AI extracts the raw numbers using advanced LLMs.
3. The engine calculates the real financial impact and flags potential risks.

## Setup Steps
1. Clone the repository: 
   ```bash
   git clone https://github.com/khanirfan18/CostPilot-AI.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Access the app locally: Open `http://localhost:3000`

**(Optional) API Keys:** You can add your API keys directly into the UI via the "API Keys" pop-up.

## Demo Usage
Click on "Try Example" in the app to instantly load a sample predatory loan agreement. 
Click **Analyze Agreement** to watch the AI break down the hidden costs and recalculate the *real* APR.
